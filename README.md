# 🧬 组织学与胚胎学 学习网站

> 🔗 **在线访问：https://maoshaoxing.github.io/embryo/**

## 功能特性

- 📚 **分类导航** — 学习大纲、教学大纲、考点汇总、名词解释、列表总结、整理笔记、重点大题、题库等 11 个模块
- 🔍 **全文搜索** — 支持模糊搜索，点击结果直接跳转
- 🌙 **暗色模式** — 一键切换，保护眼睛
- 📱 **响应式布局** — 支持桌面端与移动端
- ⚡ **无需后端** — 纯静态网站，部署在 GitHub Pages

## 数据来源

所有学习资料来自课堂 PDF 文档，经 OCR 识别整理而来。
内容覆盖：组织学与胚胎学各章节的重点名词、大题、题库、试卷等。

## 项目结构

```
embryo-site/
├── index.html       # 网站主页
├── style.css        # 样式表
├── app.js           # 交互逻辑
├── data/
│   └── content.json # 所有学习内容（自动生成）
└── README.md
```

## 本地运行

直接用浏览器打开 `index.html` 即可离线使用。

## 技术栈

- 纯 HTML/CSS/JavaScript（无框架依赖）
- Google Fonts：Noto Serif SC（思源宋体）用于正文阅读
- GitHub Pages 静态托管
